Color

Color in material design is inspired by bold hues juxtaposed with muted environments, deep shadows, and bright highlights.

Normal

To apply a background color, add the color name grey-900. To apply a text color, add the color name light-green-text-700

Sample color usage 

<div class="bas-ui-card grey-900 light-green-text-700">
    <div class="bas-ui-card-supporting-text">
        Sample color usage
    </div>
</div>

Hover

To apply a hover for background color and text color, add -hover after the name color.

Sample hover color usage 

<div class="bas-ui-card grey-900-hover light-green-text-700-hover">
    <div class="bas-ui-card-supporting-text">
        Sample hover color usage
    </div>
</div>

Sass

You can apply the color simply by extending the classes like the example below


.custom-container {
    @extend .grey-900;
}
